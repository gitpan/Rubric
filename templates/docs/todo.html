milestones

0.03

The following milestones will be reached for release 0.03, currently under development. This is not a comprehensive list: requests are welcome.

  • subclassable login

    This will let you auto-create user entries for HTTP-based user logins, map or rewrite HTTP-based logins, or retrieve authentication in other ways.

  • retrieve by date

    Queries for entries should accept dates or datetime ranges.

0.02

Release 0.02 fixed a HTTP header bug.

0.01

  • a new database can be built
  • an old database can always be upgraded
  • new users can register accounts

    New accounts are verified via email. Registration can be disabled.

  • users can change their email apress and password
  • users can create, delete, and edit their entries

    Entries have a title and tags. They might have a link, description, and a long text body. A user only has one entry per link.

  • users can copy the entries of other users
  • entries can be queried by user or tag

    Anyone can search for entries based on the user who created them or the assigned tags.

  • search results are available via rss

    Any search that can be performed via the normal HTML interface can be retrieved as RSS. The RSS links are announced on the HTML results.
